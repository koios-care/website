# Koios marketing site

Astro static site for Koios Care (Parkinson's digital medicine). No framework JS — plain Astro components + vanilla `<script>` tags for the few interactive bits.

Secondary accent is **Vibrant orange** (`#D25137` / `#E5704F`, PMS 7625 C) — from the founders' brand book (June 2026). Semantic signals (`--positive` Wood green `#2A634D`, `--amber` Fresh yellow `#DC9A61`, `--red` Vibrant orange) also come straight from the brand book.

## Commands

- `npm run dev` — dev server (http://localhost:4321)
- `npm run build` — static build to `dist/` (also the smoke test: it must pass before committing)
- `npm run preview` — serve the production build

## Sanity CMS

- Studio embedded at `/admin` (project `lw88pa2w`, dataset `production`). Schemas: `src/sanity/schemaTypes/` (teamMember, publication, quote).
- Content is fetched at build time through `src/sanity/lib/content.ts` — every loader returns `null` on empty/unreachable and pages keep hardcoded fallbacks. Never make the build depend on Sanity being up.
- New CORS origin needed per host: `npx sanity cors add <origin> --credentials`.

## Conventions

- **Design tokens live in `src/styles/global.css`** — never hardcode colors in pages; use the CSS custom properties (`--primary`, `--secondary`, `--ink-mid`, …). The brand maroon `#5B112F` is locked. `/palette` documents the system.
- Page-specific CSS goes in a `<style is:global>` block at the bottom of the page's `.astro` file (markup is partly generated by JS on some pages, so scoped styles would miss it).
- Headlines are always Didot (local "Didot" on Apple devices, GFS Didot webfont fallback — 400 + italic only, no real bold); data labels are IBM Plex Mono; body is DM Sans.
- `.reveal` elements get scroll-in animation from BaseLayout's script; add `data-delay="1..5"` for stagger. Base state is visible — never rely on JS to show content.
- Copy style: British-leaning, em-dashes, "&rsquo;" HTML entities in copy. The wellness-solution disclaimer ("not a medical device") must stay on every page that makes product claims.
- `design/` is the original Claude Design handoff (prototypes, screenshots). Reference only — don't edit, don't ship.

## Honesty constraints

This is a healthcare company. Claims on the site (57%, 96%, 40+ measures, NCT06209502, hospital names) come from the founders' material — do not invent or inflate numbers, trials, partners, or certifications. Placeholders (team, publications) are explicitly labeled as such until replaced with real data.
